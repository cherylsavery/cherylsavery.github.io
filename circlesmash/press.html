Description

Circle Smash is a fast paced casual game where circles continually appear and then grow in size. You can remove a circle by tapping it. But don't remove all the circles as you only get points when 2 circles of different colors smash into each other. The game ends when 2 circles of the same color smash together.

Features

  • Requires fast thinking and reflexes
  • Three difficulty levels
  • Support for multiple languages
  • Suitable for all ages
  • Completely FREE - no in-app purchases
